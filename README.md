📌 Project Overview

This project is an interactive Sales Dashboard built using Power BI to analyze ecommerce business performance. It provides insights into sales, profit, customer behavior, and product categories.

The dashboard helps in making data-driven decisions by visualizing key metrics in a simple and effective way.

🚀 Features
📈 Total Sales Analysis (Sum of Amount)
💰 Profit Tracking
📦 Quantity Sold Overview
🧾 Average Order Value (AOV)
🗺️ State-wise Sales Analysis
🛍️ Category-wise Distribution
📅 Monthly Profit Trends
👥 Customer-wise Sales
💳 Payment Mode Analysis
📊 Sub-category Profit Insights
🔍 Interactive Filters (Quarter & State)


📊 Dashboard Insights
Clothing category contributes the highest sales (~54%)
February shows the highest profit among months
Phones generate the highest profit among sub-categories
EMI and COD are the most used payment methods
Andhra Pradesh is a key contributing state


🛠️ Tools & Technologies
Power BI – Data visualization
Excel / CSV – Data source
DAX – Calculations & measures
